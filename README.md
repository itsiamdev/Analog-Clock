# ⏰ Analog Clock

A simple and elegant analog clock web application built using HTML, CSS, and JavaScript. This project displays the current time with smooth-moving hour, minute, and second hands, along with tick marks for a classic clock appearance.

## ✨ Features

- 🕐 Real-time analog clock display
- 🎯 Smooth hand animations
- 📏 Hour and minute tick marks
- 🌙 Dark theme with green accents
- 📱 Responsive design

## 🚀 Demo

View the live demo: [https://itsiamdev.github.io/Analog-Clock/](https://itsiamdev.github.io/Analog-Clock/)

## 🛠️ How to Run

1. 📥 Clone or download this repository
2. 🌐 Open `index.html` in your web browser
3. ⏱️ The clock will automatically display the current time and update every second

## 🛠️ Technologies Used

- 🌐 HTML5
- 🎨 CSS3
- ⚙️ JavaScript (ES6)

## 📁 Files

- `index.html` - Main HTML structure
- `style.css` - Styling and layout
- `script.js` - Clock functionality and animations
- `clock.png` - Favicon icon
